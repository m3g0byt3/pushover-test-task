PushoverSender
=======

Send notifications using [Pushover](https://pushover.net/) API.

Requirements
----------------
* iOS 9.3+
* Xcode 9.0+
* Swift 4.0+

Installation
----------------

```
git clone https://github.com/m3g0byt3/PushoverSender.git\
 && cd PushoverSender\
  && pod install\
   && open PushoverSender.xcworkspace
```
