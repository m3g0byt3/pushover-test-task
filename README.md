PushoverSender
=======

Send notifications using [Pushover](https://pushover.net/) API.

![](mockup.png)

Requirements
----------------
* iOS 10.0+
* Xcode 9.0+
* Swift 4.0+

Installation
----------------

```
git clone https://github.com/m3g0byt3/pushover-test-task.git\
 && cd pushover-test-task\
  && pod install\
   && open PushoverSender.xcworkspace
```
